1. Пользователь открывает портал помощи/входа (через Grav Slave).
2. Запрос проходит через Load Balancer → Nginx → Grav Slave, страница отдаётся пользователю.
3. Пользователь скачивает мастер автонастройки (Windows или Linux).
4. Пользователь запускает мастер локально на своём рабочем месте.
5. Мастер проводит полную диагностику конфигурации АРМ (Check).
6. Мастер автоматически исправляет найденные проблемы (Fix), где это возможно.
7. Мастер формирует и отображает итоговый отчёт.
8. Пользователь переходит в ЛК ГИИС «Электронный бюджет» через браузер.
